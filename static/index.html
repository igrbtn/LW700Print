Тип наклейки
Лента
Строки (1-8)
Код
Для QR подпись берётся из первой строки (тип «текст+QR» или «QR»).
Кабель (флажок / обёртка)
Флажок: текст у обоих концов, середина (обхват = π·диаметр) обнимает кабель. Обёртка: текст повторяется вдоль.
Патч-панель
Ячейка на порт. Строки = подписи портов (по одной на порт), иначе авто-нумерация 1..N.
Импорт из CSV (пакетная печать)
Подставляй колонки как {Имя_колонки}. Клик по заголовку - сортировка. Галочки - выбор строк (если ничего не выбрано, берётся весь фильтр). Предпросмотр -> печать.
Шаблон для агента / LLM
CSV содержит колонки из {плейсхолдеров} в строках/коде + пример. LLM/агент наполняет строки, потом загрузи CSV выше и «Печать всех строк». JSON хранит всю конфигурацию лейбла.
